# JavaScript-Basics-Learning


This repository contains my practice programs and examples covering fundamental JavaScript concepts.

## 📚 Topics Covered

- ✅ If Statement
- ✅ If...Else Statement
- ✅ Else If Ladder
- ✅ Nested If
- ✅ Ternary Operator (`? :`)
- ✅ Logical Operators
  - AND (`&&`)
  - OR (`||`)
  - NOT (`!`)
- ✅ Comparison Operators
  - Equal (`==`)
  - Strict Equal (`===`)
  - Not Equal (`!=`)
  - Strict Not Equal (`!==`)
  - Greater Than (`>`)
  - Less Than (`<`)
  - Greater Than or Equal (`>=`)
  - Less Than or Equal (`<=`)
- ✅ Switch Statement
